# 0.5 Single Page App Diagram

This diagram shows what happens when a user opens the single-page version of the notes app at  
https://studies.cs.helsinki.fi/exampleapp/spa

```mermaid
flowchart TD
A["User opens /spa page"] --> B["Browser sends GET request to server"]
B --> C["Server responds with spa.html"]
C --> D["Browser loads spa.html and requests main.css + spa.js"]
D --> E["Server sends CSS and JavaScript files"]
E --> F["Browser executes spa.js"]
F --> G["spa.js fetches JSON data (data.json) from server"]
G --> H["Server sends JSON data with existing notes"]
H --> I["Browser renders notes dynamically on the page"]
