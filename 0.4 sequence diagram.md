# 0.4 New Note Diagram

This diagram shows what happens when a user creates a new note on  
https://studies.cs.helsinki.fi/exampleapp/notes by typing a note and clicking the Save button.

```mermaid
flowchart TD
A["User writes a new note"] --> B["Clicks Save button"]
B --> C["Browser sends POST request with note data"]
C --> D["Server saves note and responds with redirect"]
D --> E["Browser reloads notes page"]
E --> F["Browser requests updated data.json"]
F --> G["Server sends updated notes list"]
G --> H["Browser displays new note on screen"]
